2. CLI Access

Use the Airflow CLI for all operations:

# List DAGs docker exec saas_platform_airflow_scheduler airflow dags list # Trigger a DAG docker exec saas_platform_airflow_scheduler airflow dags trigger analytics_pipeline_hybrid # List DAG runs docker exec saas_platform_airflow_scheduler airflow dags list-runs -d analytics_pipeline_hybrid # Run a specific task docker exec saas_platform_airflow_scheduler airflow tasks run \ analytics_pipeline_hybrid start 2025-06-20

3. Direct Pipeline Execution

Run the analytics pipeline directly (most reliable):

# Run the production pipeline python3 scripts/run_analytics_pipeline.py # View results cat logs/summary_$(date +%Y%m%d).txt
